Async JavaScript

// Promises
function fetchUser(id) {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      if (id > 0) resolve({ id, name: "Alice" });
      else reject(new Error("Invalid ID"));
    }, 1000);
  });
}

// .then() chaining
fetchUser(1)
  .then(user => fetchPosts(user.id))
  .then(posts => console.log(posts))
  .catch(err => console.error(err));

// async/await (preferred)
async function loadDashboard() {
  try {
    const user = await fetchUser(1);
    const posts = await fetchPosts(user.id);
    render(user, posts);
  } catch (err) {
    showError(err.message);
  }
}

// Parallel execution
const [users, posts] = await Promise.all([
  fetchUsers(),
  fetchPosts()
]);

// Promise.allSettled - doesn't short-circuit on rejection
const results = await Promise.allSettled([fetch1, fetch2, fetch3]);

Fetch API

// GET request
const response = await fetch("/api/users");
if (!response.ok) throw new Error(`HTTP ${response.status}`);
const users = await response.json();

// POST request
const newUser = await fetch("/api/users", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ name: "Alice", email: "a@b.com" })
});

// With error handling
async function api(url, options = {}) {
  const res = await fetch(url, {
    headers: { "Content-Type": "application/json", ...options.headers },
    ...options
  });
  if (!res.ok) {
    const error = await res.json().catch(() => ({}));
    throw new Error(error.message || `HTTP ${res.status}`);
  }
  return res.json();
}

// AbortController (cancel requests)
const controller = new AbortController();
fetch("/api/data", { signal: controller.signal });
controller.abort();  // cancel the request

Classes

class Animal {
  #sound;  // private field

  constructor(name, sound) {
    this.name = name;
    this.#sound = sound;
  }

  speak() {
    return `${this.name} says ${this.#sound}`;
  }

  get info() {
    return `${this.name} (${this.constructor.name})`;
  }

  static create(name, sound) {
    return new Animal(name, sound);
  }
}

class Dog extends Animal {
  constructor(name, breed) {
    super(name, "Woof");
    this.breed = breed;
  }

  fetch(item) {
    return `${this.name} fetches ${item}!`;
  }
}

const rex = new Dog("Rex", "Labrador");
rex.speak();      // "Rex says Woof"
rex.fetch("ball"); // "Rex fetches ball!"
rex instanceof Animal;  // true

Web APIs

Intersection Observer

Detect when elements enter/exit viewport. Perfect for lazy loading and infinite scroll.

Web Workers

Run CPU-intensive tasks in background threads. No UI blocking. Communicate via postMessage.

Service Workers

Proxy between app and network. Enable offline support, caching, push notifications (PWA).

WebSockets

Full-duplex real-time communication. new WebSocket(url). Events: open, message, close.

ResizeObserver

Watch element size changes. Better than window resize events for component-level responsiveness.

Clipboard API

navigator.clipboard.writeText(text). Modern async replacement for execCommand.

Best Practices

const Use const by default, let only when reassignment needed
=== Always use strict equality (=== not ==)
Arrow Prefer arrow functions for callbacks and closures
Async Use async/await over .then() chains
ESLint Lint your code. Popular config: eslint-config-airbnb
Modules Use ES modules (import/export) everywhere
Security Avoid innerHTML (XSS), validate user input
?. ?? Use optional chaining and nullish coalescing
