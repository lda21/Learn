Command Palette & Navigation

Command Palette Prefixes

  • > — commands (default)
  • @ — go to symbol in file
  • @: — go to symbol by category
  • # — go to symbol in workspace
  • : — go to line number
  • No prefix — quick open file

Navigation

  • F12 — Go to Definition
  • Alt+F12 — Peek Definition
  • Shift+F12 — Find All References
  • Ctrl+Tab — Switch open editors
  • Ctrl+G — Go to line
  • Breadcrumbs — click path segments

Editing Power

Multi-Cursor

Ctrl+D select next match. Ctrl+Shift+L select ALL matches. Alt+Click add cursor anywhere.

Find & Replace

Ctrl+H with regex support. Use capture groups $1 for powerful transformations.

Emmet

Type div.container>ul>li*5 then Tab to expand HTML. Built-in, no extension needed.

Code Folding

Ctrl+Shift+[ fold, Ctrl+Shift+] unfold. Ctrl+K Ctrl+0 fold all.

// Emmet example: ul>li.item{Item $}*3 + Tab
<ul>
  <li class="item">Item 1</li>
  <li class="item">Item 2</li>
  <li class="item">Item 3</li>
</ul>

Integrated Terminal

Features

  • Ctrl+` toggle terminal panel
  • Multiple terminals (split with icon)
  • Rename, color-code terminals
  • Shell integration (bash, zsh, PowerShell)
  • Click file paths to open in editor
  • Link detection for URLs and errors

Profiles

  • Set default shell per OS
  • Create custom profiles (bash, zsh, node)
  • Custom environment variables per profile
  • Ctrl+Shift+` new terminal
  • Tasks run in terminal automatically

Tip: Use Ctrl+Shift+5 to split terminal. Run server on left, tests on right.

Git Integration

Source Control Panel

  • View changes, stage files with +
  • Inline diff view (click any file)
  • Commit with message box
  • Branch switching in status bar
  • Sync, push, pull buttons

Merge Conflicts

  • Visual merge editor (3-way)
  • Accept Current / Incoming / Both
  • Inline conflict markers with buttons
  • Compare side by side

GitLens extension adds: blame annotations, file history, line history, commit graph, and remote repository browsing.

Essential Extensions

Prettier

Auto-format on save. Supports JS, TS, CSS, HTML, JSON, Markdown.

ESLint

JavaScript/TypeScript linting with auto-fix. Integrates with Prettier.

GitLens

Git supercharged: blame, history, stash, commit graph, remote repos.

Live Server

Local dev server with hot reload for static HTML/CSS/JS files.

REST Client

Send HTTP requests from .http files. Like Postman in your editor.

Error Lens

Inline error/warning messages right next to the code. No need to hover.

Debugging

Breakpoints

  • Click gutter to set breakpoint
  • Conditional — break when expression is true
  • Logpoints — log without stopping
  • Hit Count — break after N hits
  • Exception breakpoints (caught/uncaught)

Debug Features

  • Variables panel (local, closure, global)
  • Watch expressions
  • Call stack navigation
  • Debug console (evaluate expressions)
  • Step over / into / out controls

Settings & Configuration

Settings Scope

  • User — global, all projects
  • Workspace — per-project (.vscode/)
  • Workspace overrides User settings
  • GUI or JSON (settings.json)

Must-Have Settings

{
  "editor.formatOnSave": true,
  "editor.tabSize": 2,
  "editor.minimap.enabled": false,
  "editor.bracketPairColorization.enabled": true,
  "files.autoSave": "onFocusChange",
  "editor.stickyScroll.enabled": true
}

Settings Sync: Sign in with GitHub/Microsoft to sync settings, keybindings, extensions, and snippets across machines.

Workspaces & Tasks

Multi-Root Workspaces

  • Open multiple folders in one window
  • Save as .code-workspace file
  • Per-folder settings and extensions
  • Useful for monorepos and microservices

Tasks (tasks.json)

{
  "label": "Build",
  "type": "npm",
  "script": "build",
  "group": {
    "kind": "build",
    "isDefault": true
  },
  "problemMatcher": ["$tsc"]
}

Ctrl+Shift+B runs default build task.
